## Write Basic Git Commands Below

- Clone a Repository:
```
$ git clone <url>
```

- Check Status:
```
$ git status
```

- Stage (Add):
```
$ git stage [-A]
```

- Commit:
```
$ git commit -m "commit message"
```

## Write Basic GitHub Commands Below

- Push:
```
$ git push origin master
```

- Check Remote:
```
$ git remote -v
```
